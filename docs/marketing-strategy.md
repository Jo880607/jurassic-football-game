## 🎬 5. 게임 소개 영상 시나리오

### 메인 트레일러 (2분 버전)

**[Scene 1: 0-15초]**
```
- 검은 화면에서 공룡 발자국 소리
- "6500만년 전, 그들이 돌아왔다"
- T-Rex 포효 + 축구공이 굴러오는 소리
```

**[Scene 2: 15-45초]**  
```
- 다양한 공룡들이 축구복을 입고 등장
- 빠른 컷 편집으로 각 공룡의 특수 스킬 시연
- BGM: 웅장한 오케스트라 + 현대적 비트
- 나레이션: "12종의 전설 공룡들과 함께"
```

**[Scene 3: 45-75초]**
```
- 실제 게임플레이 화면 
- 카드 드래그 → 공룡 소환 → 골 장면
- "간단한 조작, 무한한 전략"
```

**[Scene 4: 75-90초]**
```
- 다양한 스타디움과 환경
- "화산구장에서 빙하구장까지"
- 멀티플레이 매치 장면
```

**[Scene 5: 90-120초]**
```
- 로고 등장 + 출시일 안내
- "지금 사전등록하고 골든 렉스를 받아가세요!"
- 앱스토어/구글플레이 로고
```
